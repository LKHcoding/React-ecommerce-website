# React E-commerce(프로젝트 진행중)

### **React.js + MongoDB + node.js 를 활용한 E-commerce website**

- 로그인, 로그아웃, 회원가입 기능 - 완료
- 비밀번호 암호화 및 쿠키 세션관리 - 완료
- 회원가입 내 주소 검색(daum api) - 완료
- 관리자 페이지(유저 관리, 상품 등록 및 관리 등) - 개발중
- 인터렉티브 디자인(Index페이지 및 소개페이지) - 개발중
- 마이페이지 - 개발중
- 상품상세페이지(리뷰댓글포함)
- 구매페이지
- 장바구니
